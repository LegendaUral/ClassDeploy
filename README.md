# ClassDeploy — управление учебным классом

Инструмент дистанционного управления учебными ПК: трансляция экранов, установка ПО, блокировка, управление мышью, звук и трансляция экрана.

## Структура проекта

```
classdeploy/
├── shared/            # Общий код: протокол, конфиг
├── server/            # Сервер (PyQt6 GUI + WebSocket)
├── agent/             # Агент (Windows-служба)
├── start_server.bat   # Запуск сервера
├── install_agent.bat  # Установка агента на ПК ученика
├── uninstall_agent.bat
├── build_agent.bat    # Сборка EXE агента
└── requirements.txt
```

## Быстрый старт

### 1. Сервер (ПК учителя)

```bat
pip install -r requirements.txt
start_server.bat
```

### 2. Агент (ПК ученика)

**Шаг 1.** Соберите EXE на ПК учителя:
```bat
build_agent.bat
```

**Шаг 2.** Скопируйте `ClassDeployAgent.exe` на ПК ученика.

**Шаг 3.** Запустите `install_agent.bat` от администратора — введите IP учительского ПК.

Служба установится и запустится автоматически. Она работает от учётной записи SYSTEM — это значит, что агент активен при **любом** вошедшем пользователе.

### 3. Настройка адреса сервера

Агент ищет IP сервера в:
1. Переменная окружения `CLASS_DEPLOY_SERVER`
2. Файл `C:\ProgramData\ClassDeploy\server.txt` (создаётся при установке)
3. `127.0.0.1` (заглушка для теста)

### 4. Проверить статус службы

```bat
sc query ClassDeployAgent
```

Лог агента: `C:\ProgramData\ClassDeploy\logs\agent.log`

## Возможности

| Функция | Описание |
|---|---|
| 📦 Установка | Drag-and-drop .msi/.exe/.zip, автокликер |
| 🖥 Трансляция | Сетка экранов, двойной клик — полный экран |
| 🖱 Управление | В полноэкранной трансляции доступно управление только мышью |
| 🔒 Блокировка | Чёрный экран с сообщением |
| 😱 Скример | Полноэкранное изображение (своя картинка) |
| 💬 Сообщения | MessageBox на ПК ученика |
| 🔊 Звук | Громкость, мут/анмут |
| 🎵 Аудио | Проигрывание звука или музыки на всех выбранных ПК |
| ⌨ Комбинации | Win+D, Ctrl+Alt+Del, Alt+F4 и другие |
| 📵 Ограничения | Блокировка сайтов и приложений |
| ⏻ Питание | Выключение, перезагрузка, завершение сеанса |
| 🔍 Процессы | Диспетчер задач с именами, путями, поиском |

## Порт

WebSocket: **8765** (TCP). Откройте на брандмауэре ПК учителя (входящие).

## Зависимости

- Python 3.10+
- `websockets`, `PyQt6`, `mss`, `Pillow`
- `pywin32`, `pywinauto`, `pycaw` (только Windows, агент)


## Новое в этой сборке

- Агент ставится в `C:\ProgramData\ClassDeploy\Agent`, что снижает риск ошибок доступа на части ПК.
- Для агента добавлен watchdog: после аварийного завершения процесс автоматически перезапускается.
- В окне трансляции удалён захват клавиатуры, оставлено только управление мышью.
- В сервер добавлено удалённое проигрывание аудиофайла на выбранных компьютерах.
